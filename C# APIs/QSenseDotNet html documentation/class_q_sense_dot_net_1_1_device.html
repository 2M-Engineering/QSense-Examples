◆ ReadMemory()

void QSenseDotNet.Device.ReadMemory ( )

Reads QSense Motion Device memory if BLE is connected and not streaming.

This method first verifies the BLE connection, then checks the current device state. If the device is connected and not actively streaming, it triggers a read operation on the memory control area using bleApi.ReadMemory.

◆ Reset()

void QSenseDotNet.Device.Reset ( )

Sets Sensor Config.

This method sets accelerometer, gyroscope sensitivity as well as sampling rate and the number of raw samples that are buffered in each stream packet

Parameters
accSensSensitivityAcc type input getting converted to bits and stored in the device memory as accelerometer sensitivity.
gyrSensSensitivityGyr type input getting converted to bits and stored in the device memory as gyroscope sensitivity.
sampRateSamplingRate type input getting converted to bits and stored in the device memory as sampling rate.
bufferingInteger type input getting converted to bits and stored in the device memory as the number of raw samples that are buffered in each stream packet.
